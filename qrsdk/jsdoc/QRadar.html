rest(args)

Calls a REST method using an XMLHttpRequest.
Parameters:
Name Type Description
args Object
Properties
Name Type Attributes Default Description
httpMethod String The HTTP method to use (GET/PUT/POST/DELETE).
path String The path to the REST endpoint.
  • To call a QRadar REST API, path must start with "/api".
  • To call a REST endpoint in your application, path must start with "/application".
  • Any other path must be a fully-qualified URL, otherwise the function behaviour is undefined.
body String <optional>
 The data to POST or PUT.
onComplete function <optional>
 Callback function to be invoked when the REST request finishes. The function can access the XMLHttpRequest using 'this'.
onError function <optional>
 Callback function to be invoked if the REST request fails to complete.
headers Array <optional>
 Headers to be supplied with the REST request. Each array entry should be a JSON object with "name" and "value" properties.
contentType String <optional>
 "application/json" MIME type of a POST or PUT request. Default value is used only if Content-Type is not supplied in args.headers.
timeout Number <optional>
 HTTP timeout, in milliseconds, to be supplied with an asynchronous REST request. If args.async is false, the timeout is ignored.
async boolean <optional>
 true Set to false to make a synchronous request. WARNING: this is not recommended.
